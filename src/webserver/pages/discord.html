Introduction

New to discord, or want to know how TripSits discord works in detail? This is the page for you!

Definitions

Guild
What discord calls a collection of discord channels.
EG: Instead of joining an IRC server, you join a discord guild.

Role
A set of permissions.
EG: People can be given the Moderator role, which gives them the ability to kick/ban users.

Channel
Another name for a chat room.
EG: The lounge channel is a fun place to chat

Member / User
The someone who belongs to a guild
EG: I like this member, theyâ€™re a positive influence

Experience / Level 2 / VIP
As users talk they generate "experience points" that determine their â€œlevelâ€, just like a videogame. Different levels have different rewards.
See Levels

Embed
A type of message sent by bots that allows creative formatting
EG: Normal users send messages but when i do the /about command the formatted message is an embed

Ephemeral
This describes a type of message from the bot that can only be seen by the user who did the command.
EG: If you do /idose you will see the following message, note the final words that appear at the very bottom:

Thread-only mode (forum mode)
When a channel does not allow you to type, but you can make a new thread to post a message that can be responded to like a normal chat room.
EG: #drug-questions room does not allow messages, but allows the user to click + and make a new thread to ask a question

Discord Specific Policies

Threads
Use the little + icon in most channels to make a temporary channel (thread) to discuss a topic.
This allows us to keep our thoughts organized instead of all in one big channel. Please use threads as much as possible!

Text-in-voice-channels
Voice channels have a partner text channel that can be used to type instead of talk!

Roles
Things to know about Roles:

Roles exist in a hierarchy so that you will gain permissions/customization from the highest role you have.
IE: All staff members will always have the given icon next to their name
EG: If you're a Moderator, you will always have a ğŸ›¡ï¸ next to your name, even if you pick one of the roles that give a different icon.

The only roles with colors are the color-roles

EG: As an Admin you don't need to have the Purple color, you can pick whatever you want to express yourself!

Mindset roles are higher than functional roles
IE: As a user, the mindset you pick will override whatever functional role you have
EG: If you have the Helper role you'll have a ğŸ•â€ğŸ¦º next to your name, but if you pick the Drunk role the ğŸº will take priority.
