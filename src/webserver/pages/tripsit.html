In conclusion: People who need help can click the "I need assistance" button in the 😇│tripsit room. They enter their question and a private thread is created in 😇│tripsit. This is a self-contained channel so that trolls/randoms cannot access what they're saying. When the user no longer needs help they can click a button and return to normal. The user has 24 hours to follow-up on the thread before it's archived, and we can re-use the thread for a week before it's deleted.
